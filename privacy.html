Политика конфиденциальности

Дата последнего обновления: 1 января 2025 года

Linkora eSIM («мы») серьёзно относится к защите ваших персональных данных. Настоящая политика описывает, какие данные мы собираем, как используем и защищаем их.

1. Какие данные мы собираем

Мы собираем минимально необходимые данные для оказания услуги:

  • Telegram ID — для идентификации аккаунта и доставки eSIM
  • Telegram username — для связи в рамках поддержки
  • Данные заказа — выбранный тариф, сумма оплаты, статус активации
  • Подтверждение платежа — номер транзакции (данные карты не хранятся)

2. Как мы используем данные

  • Для обработки заказа и доставки QR-кода eSIM
  • Для оказания технической поддержки
  • Для отправки уведомлений о заказе через Telegram
  • Для предотвращения мошенничества и злоупотреблений

3. Хранение данных

Данные хранятся в защищённой базе данных. Записи о заказах хранятся до 12 месяцев для целей поддержки и разрешения споров. Вы можете запросить удаление данных в любое время через поддержку.

4. Передача данных третьим лицам

Мы передаём ограниченные данные следующим партнёрам:

  • Поставщик eSIM (eSIM Cards) — для выполнения заказа
  • Платёжные системы — Telegram Stars, CryptoBot — только в объёме, необходимом для обработки платежа

Мы не продаём ваши данные третьим лицам. Данные не используются в рекламных целях.

5. Данные Telegram

Наш сервис работает через Telegram. К вашему использованию платформы Telegram применяется Политика конфиденциальности Telegram. Мы получаем только те данные, которые Telegram предоставляет ботам (ID, username, имя).

6. Ваши права

Вы имеете право:

  • Запросить доступ к данным, которые мы храним о вас
  • Потребовать исправления неточных данных
  • Потребовать удаления ваших данных
  • Отозвать согласие на обработку данных в любое время

Для реализации прав обратитесь: @esimruu_bot

7. Безопасность

Мы применяем технические и организационные меры для защиты ваших данных. Тем не менее ни одна передача данных через интернет не является на 100% защищённой.

8. Файлы cookie

Наш сайт не использует файлы cookie для отслеживания или рекламы.

9. Дети

Наш сервис не предназначен для лиц моложе 18 лет. Мы не собираем данные несовершеннолетних намеренно.

10. Изменения политики

Мы можем периодически обновлять настоящую политику. Изменения публикуются на этой странице с обновлённой датой.

11. Контакты

Вопросы о конфиденциальности: @esimruu_bot
